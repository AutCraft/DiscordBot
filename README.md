# Discord Bot

A customizable Discord Bot built with discord.js.

## Features
- Discord Bot functionality using discord.js
- Music playback features using `discord-player`, `play-dl`, and `ytdl-core`
- Slash commands support

## Prerequisites
- Node.js >= 16.x

## Installation

1. Clone the repository
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file in the root directory with the following variables:
   ```env
   DISCORD_TOKEN=your_discord_bot_token
   TENORKEY=your_tenor_api_key
   PREFIX=>
   ```

## Running the Bot

To start the bot, run:
```bash
npm start
```
